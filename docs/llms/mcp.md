This document will contain documentation for mdpress MCP (Model Context Protocol) server integration.

---

## Overview

MCP integration will enable AI assistants and language models to interact directly with mdpress through standardized tool interfaces.

---

## Planned Capabilities

When MCP integration is complete, it will enable:

- **Direct rendering** - Render Markdown to PDF from AI assistants
- **Template discovery** - List and select available templates
- **Data validation** - Validate data files before rendering
- **Configuration access** - Read and apply mdpress configuration
- **Preview generation** - Generate previews without full PDF output
